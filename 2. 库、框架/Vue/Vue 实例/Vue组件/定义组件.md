# 定义组件
- 组件名称使用了驼峰命名的话，在引用组件的时候，要**把大写的驼峰改为小写的字母**，同时，两个单词之间使用 **-** 连接
- 不论是全局还是局部创建出来的组件，组件的`template`属性指向的模板内容，**有且只能有唯一的一个根元素**
# 定义全局组件

`Vue.component('组件的名称'，创建出来的组件模板对象)`

## 方式一 先使用Vue.extend创建组件模板对象
1. 使用 Vue.extend 来创建全局的 Vue 组件,通过 template 属性，指定库组件要展示的HTML结构
2. 使用Vue.component('组件的名称'，创建出来的组件模板对象) 
``` html
<div id="app">
    <my-com></my-com>
</div>

<script>
    var com1 =  Vue.extend({
        template:'<h3>这是使用 Vue.extend 创建的组件</h3>'
    }) 
    Vue.component('myCom',com1)
</script>
```
也可以写在一起

```html
<div id="app">
    <my-com></my-com>
</div>
<script>
    Vue.component('myCom',Vue.extend({
        template:'<h3>这是使用 Vue.extend 创建的组件</h3>'
    }))
</script>
```

## 方式二 直接创建组件
```html
<div id="app">
    <my-com></my-com>
</div>

<script>
    Vue.component('myCom',{
        template:'<h3>直接使用 Vue.component 创建的组件</h3>'
    })
</script>
```

## 方式三 使用 template 标签
- 在被控制的 #app 外面使用 template 元素，定义组件的模板结构 
- 这个方式，有代码的智能提示和高亮
```html
<div id="app">
    <my-com></my-com>
</div>

<template id="tmp1">
    <div>
        <h5>在外部定义的组件 template 结构，</h5>
    </div>
</template>

<script>
    Vue.component('myCom',{
        template:'#tmp1'
    })
</script>
```

# 定义局部组件  
```html
<div id="app">
    <my-header></my-header>
</div>

<script>
    new Vue({
        el:"#app",
        components:{ //定义实例内部私有组件
            'myHeader':{
                template:'<h4>这是私有的 login 组件</h4>'
            }
        }
    })
</script>
```
# 注意
## 禁止data引用赋值

引用赋值：值只有一份，会使所有组件实例复用一份data
```javascript
data:{
    name:"liunan"
}
```
可以通过将data设置为函数返回值，这样每复用一次组件，就会返回一个新的data，类似于给每个组件实例创建了一个私有空间，让每个组件维护各自的数据。
```javascript
data(){
    return{
        name:"liunan"
    }
}
```
