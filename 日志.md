6. 错误在哪里? 
```
function functions(flag) {
   var getvalue=null;
    if (flag) {
      getValue = function(){ return 'a'; }
    } else {
      getValue = function() { return 'b'; }
    }

    return getValue();
}
```
- 对于**函数声明**，解析器会率先读取并让其在执行任何代码前可用->代码还没运行，getValue函数声明就被读取了
- 对于**函数表达式**，当解析器执行到它所在的代码行时，才会被真正解释执行
