## 19. 算法题1
对一个给定数进行千位数划分，比如 12345678，结果为 12,345,678
```
function format(num){
  var str = num+""
  var n=0,arr=[],k=0
  for(var i=str.length-1;i>=0;i--){
    arr[n]=str[i]
    k++
    n++
    if(k%3==0){
      arr[n]=","
      n++
    }
  }
  return arr.reverse()
}
```
